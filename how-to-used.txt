â“ 9. FREQUENTLY ASKED QUESTIONS
================================================================================

Q: Can I use this with non-Gmail accounts?
A: Currently optimized for Gmail. Other providers may work if they support
   IMAP/SMTP with same settings.

Q: How often does it check for new emails?
A: Every 60 seconds when auto-responder is running.

Q: Will it reply to spam emails?
A: Yes, it replies to all unread emails in inbox. Gmail's spam filter
   should prevent most spam from reaching inbox.

Q: Can I customize the auto-reply message?
A: Yes! Edit the AUTO_REPLY_HTML variable in the Python code.

Q: Does it work when my computer is off?
A: No, the application must be running. Consider using a cloud server
   for 24/7 operation.

Q: How many emails can it handle?
A: Gmail has sending limits:
   - Free accounts: ~500 emails/day
   - Workspace accounts: ~2000 emails/day

Q: Can I run multiple instances?
A: Not recommended. Use one instance per email account.

Q: What happens if I close the application?
A: Auto-responder stops immediately. Unsent replies are lost.

Q: Is my password stored anywhere?
A: No! Password is only kept in memory during the session.
   It's cleared when you logout or close the app.

Q: Can I schedule auto-responder to start/stop automatically?
A: Not currently. You must manually start/stop it.
